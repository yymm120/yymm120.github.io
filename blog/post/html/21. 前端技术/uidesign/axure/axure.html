                                                                                                                                                                      如果你想将你的原型发布到 Axure Cloud 以外的地方，你可以通过导出原型的 HTML 文件，找到 Publish（发布） → Generate HTML Files（生成 HTML 文件） ，然后根据需要修改相关设置，最后点击 Publish（发布） 。

                                                                                                                                                                      你可以把导出的 HTML 文件分发给任何人查看。

                                                                                                                                                                      格式刷

                                                                                                                                                                      推/拉 #

                                                                                                                                                                      当显示或隐藏一个 Widgets（元件），或改变面板状态时，你可以选择在下面或右边推拉 Widgets（元件），以便为显示的 Widgets（元件）腾出空间，或关闭被隐藏的 Widgets（元件）腾出的空间。

                                                                                                                                                                      对于组，只要组的边界框落在移动区域内，整个组就会被推或拉，移动区域是指被显示或隐藏的参考组的左上方以下的区域。

                                                                                                                                                                      如果一个组的边框不在移动区域内，那么整个组将不会被推或拉。相反，只有组内处于移动区域的个别元件会被推或拉。

                                                                                                                                                                      组上的交互与其包含的Widgets（元件）上的交互 #

                                                                                                                                                                      组有一些与它们所包含的元件可能有的相同的事件（例如点击或轻触），这有可能导致冲突。如果一个组和它所包含的一个Widgets（元件）在同一个事件上设置了动作，Widgets（元件）的动作将被执行，而不是组的。

                                                                                                                                                                      添加本地库文件 #

                                                                                                                                                                      要将本地存储的Widgets（元件）库文件添加到 libraries（元件库） 面板中使用，请点击面板顶部的 更多元件库 图标。在出现的文件浏览器中，找到你想要的 .rplib 文件，并选择它以将其添加到面板中。

                                                                                                                                                                      创建元件库 #

                                                                                                                                                                      找到 File（文件） → New Library 新元件库 ，创建一个新的元件库。要把你的元件库变成一个团队项目，请遵循创建和分享团队项目 (opens new window)文章中的步骤。

                                                                                                                                                                      将参考页分配给形状可以做三件事:

                                                                                                                                                                      • 形状上的文字被设置为该页的名称。
                                                                                                                                                                      • 如果页面被重新命名，该形状的文本会自动更新。
                                                                                                                                                                      • 在浏览器中点击该形状就可以打开该页面。

                                                                                                                                                                      要将参考页分配给形状元件，在 Interaction 面板的 Shape Properties 部分点击 Show All ，然后点击 Reference Page 。

                                                                                                                                                                      形状转换 #

                                                                                                                                                                      你可以对一个或多个选定的形状元件应用一些转换。右键形状，然后进入 Transform Shape ，以访问这些选项。

                                                                                                                                                                      Flip Horizontal/Vertical: 在 Y 轴（水平方向）或 X 轴（垂直方向）上翻转形状。

                                                                                                                                                                      Unite 联集: 将多个形状合并为一个路径

                                                                                                                                                                      Subtract 减去顶层: 将一个或多个形状从另一个形状中减去。前面的形状将从最后面的形状中减去。

                                                                                                                                                                      Intersect 交集: 只保留两个或多个图形的相交部分

                                                                                                                                                                      Exclude 差集: 将两个或多个形状连接在一起，排除任何重叠的部分。重叠的区域被消除，每个形状的剩余部分被保留在一个可能有多个路径的形状中。

                                                                                                                                                                      Combine 合并: 将两个或多个形状合并成一个形状，并保留每个原始路径（与 Unite 相反）。

                                                                                                                                                                      Break Apart 分解: 将以前组合的形状分解成独立的形状

                                                                                                                                                                      Curve/Sharpen All Points 弯曲/锐化所有点: 弯曲或锐化所选形状的所有矢量点。

                                                                                                                                                                      基本部件

                                                                                                                                                                      形状元件

                                                                                                                                                                      形状元件位于顶部工具栏。

                                                                                                                                                                      复选框元件

                                                                                                                                                                      用于在表单输入中, 为用户提供一个二元选择。在一些设计模式中，复选框有第三种 “不确定” 状态，即既不勾选也不取消选中。该元件不支持"不确定"状态，需要用形状元件自己实现。

                                                                                                                                                                      菜单元件

                                                                                                                                                                      基础菜单元件被用来建立导航菜单。

                                                                                                                                                                      下拉列表元件

                                                                                                                                                                      下拉列表或叫做下拉菜单 -- 是一个可切换的列表，它包含几个选项，每次只能选择其中一个。下拉列表元件位于 Default 元件库的 Forms 部分。

                                                                                                                                                                      动态面板元件

                                                                                                                                                                      一个动态面板是一个容器，它可以容纳其他元件。

                                                                                                                                                                      一个动态面板可以有一个或多个状态，而且在同一时间只有一个状态是可见的。可见状态可以通过 Set Panel State 动作动态设置。

                                                                                                                                                                      动态面板的独特之处在于，它们是唯一可以在浏览器中拖动或滑动的元件类型。它们也是唯一可以被钉在浏览器窗口固定位置的元件类型。

                                                                                                                                                                      动态面板特定的事件和行动

                                                                                                                                                                      动态面板具有其他元件所不具备的一些特殊功能：它们可以被拖动、滑动和滚动，而且一个面板的活动状态（可见的那个）可以动态地改变。你可以通过动态面板特有的事件（ Dragged,Swiped Up 等）和动作（ Set Panel State,move with drag, 等）来驾驭这些能力。

                                                                                                                                                                      热区元件

                                                                                                                                                                      内联Frame元件

                                                                                                                                                                      内联 Frame 元件允许你将 HTML、视频、音频和其他媒体文件嵌入你的 Axure RP 原型中。

                                                                                                                                                                      线路元件

                                                                                                                                                                      选择框元件

                                                                                                                                                                      选择框元件显示一个选项列表，用户可以从中选择一个或几个选项。选择框元件位于 Default 元件库的 Forms 部分

                                                                                                                                                                      单选按钮元件

                                                                                                                                                                      单选按钮是一种允许用户从多个选项中选择一个的表单控件。选择一个单选按钮将自动取消对同一组中任何先前选定的单选按钮的选择。

                                                                                                                                                                      快照元件

                                                                                                                                                                      快照元件可以理解为在你的原型中的页面上显示其他原型（页面和组件）的屏幕截图。每当引用的页面或组件被更新时，快照元件中的图像就会自动更新以适应。快照元件对于绘制用户流程图来说非常方便。

                                                                                                                                                                      表格元件

                                                                                                                                                                      文本字段和文本区元件

                                                                                                                                                                      你可以给文本字段元件不同的输入类型，以显示它们在用户输入表中的功能。

                                                                                                                                                                      • Text: 用于基本文本输入的默认设置
                                                                                                                                                                      • Password: 文本在输入时被屏蔽
                                                                                                                                                                      • Email: 可能会在移动设备上提示电子邮件键盘
                                                                                                                                                                      • Number: 只接受数字输入，可能在移动设备上提示数字键盘

                                                                                                                                                                      树形元件

                                                                                                                                                                      树形元件最常被用来模拟文件浏览器和可视化其他分层结构。点击树形元件的各个节点可以在页面上显示不同的元件或打开项目中的不同页面。

                                                                                                                                                                      低保真模式

                                                                                                                                                                      页面被转换为灰度显示效果，同时所有字体都被替换为手写字体，让界面看起来更加粗糙。

                                                                                                                                                                      仅关注设计的用户体验，而不是其视觉效果。

                                                                                                                                                                      组件

                                                                                                                                                                      组件是元件的合集，你可以在一个集中的地方维护这些元件，并在整个项目中重复使用。你对主组件所做的修改会自动推送到项目中的每个组件实例。

                                                                                                                                                                      创建组件

                                                                                                                                                                      • 选中需要的元件, 右键, 选择创建组件
                                                                                                                                                                      • 在界面的左下角, 母版中创建组件

                                                                                                                                                                      组件视图

                                                                                                                                                                      双击组件, 进入组件的编辑模式, 然后在 Style 面板中点击 Add Component Views ，打开 Component Views 对话框，在这里你将创建和管理你的视图。
